# Hakuu's Crosshair

A modern crosshair mod for PAYDAY 2 with hit feedback, laser-follow aiming, and target-based color changing.

## Features

### Hit Marker
Visual feedback when hitting enemies:
- **White** - Normal hit
- **Red (larger)** - Headshot
- **Gold (larger)** - Critical hit
- **Bright Red (largest)** - Kill confirmed

### Laser Follow
The crosshair follows your weapon's actual aim point:
- Syncs with weapon laser position
- Smooth tracking with adjustable sensitivity
- Returns to center when not holding a weapon

### Target-Based Colors
Crosshair color changes based on what you're aiming at:
| Target | Color |
|--------|-------|
| Default | White |
| Enemy | Red |
| Special Enemy | Orange |
| Civilian | Yellow |
| Teammate | Green |
| Hostage | Light Blue |
| Interactable | Cyan |
| Camera | Purple |

### Crosshair Modes
- **Auto** - Cross when shooting, dot during stealth/interaction/melee
- **Dot** - Always show dot
- **Cross** - Always show cross
- **Hide when ADS** - Optionally hide crosshair when aiming down sights

### Dynamic Expansion
- Crosshair expands when firing
- Automatically contracts when idle
- Simulates recoil feedback

### Fully Customizable
- Crosshair length, thickness, and gap
- Dot size
- Opacity
- Outline toggle
- Laser follow sensitivity
- Hit marker size and duration

## Installation

1. Install [SuperBLT](https://superblt.znix.xyz/)
2. Download this mod
3. Copy the folder to `PAYDAY 2/mods/`
4. Launch the game

## Configuration

### In-Game Menu
1. Go to main menu
2. Select **Options** > **Mod Options** > **Hakuu's Crosshair**
3. Adjust settings as desired

### Keybinds
You can bind a key to toggle the crosshair:
1. Main Menu > **Options** > **Mod Keybinds**
2. Find **Toggle Crosshair**
3. Set your preferred key

## Settings Reference

| Setting | Description | Default |
|---------|-------------|---------|
| Enable Crosshair | Toggle custom crosshair | On |
| Crosshair Mode | Auto/Dot/Cross | Auto |
| Hide When ADS | Hide when aiming down sights | On |
| Dot Size | Size of dot | 4 |
| Crosshair Length | Length of crosshair lines | 8 |
| Crosshair Thickness | Thickness of lines | 2 |
| Crosshair Gap | Center gap | 4 |
| Show Outline | Black outline around crosshair | On |
| Laser Follow | Crosshair follows weapon aim point | On |
| Follow Sensitivity | How closely crosshair follows | 1.0 |
| Enable Hitmarker | Show hit confirmation | On |
| Dynamic Expansion | Crosshair expands when firing | On |

## Compatibility

- PAYDAY 2 (latest version)
- Requires SuperBLT
- Compatible with most HUD mods
- VR mode not supported
